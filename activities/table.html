When do you use a table?

You should use a table when you need to show organized data, compare items side by side, or display large amounts of information clearly.

When do you not use a table?

Avoid using tables for narrative content or simple lists that do not need complex structure.

When do you need to do extra work for table accessibility?

For table accessibility, ensure headers are used properly, add captions and summaries for context, use the scope attribute to define relationships, and make sure the table is readable by screen readers.

Why don't we use the border attribute?

The border attribute was part of older HTML specifications but is now deprecated in favor of CSS. Using CSS to style borders is more versatile and separates content from presentation, adhering to modern web development practices.

What is the easiest way to color every other row a different color?

You can use the CSS :nth-child selector to easily color every other row.
