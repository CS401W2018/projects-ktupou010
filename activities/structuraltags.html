Structural Tags (HEADER, NAV, MAIN, FOOTER)

What goes in the header?

- The header element typically contains introductory content, such as:

  • Logo
  • Site title
  • Tag line
  • Navigation link

What goes in the Nav? Can the Nav go in the Header?

- The navigation element contains links to other pages or sections within the website or application. It can contain:

  • Link to other sections on the website
  • Link to sub section
  • Link to an external pages

The navigation element can be placed inside the header element, but it's not a requirement. It can also be placed elsewhere in the document, such as in the main or footer elements, depending on the design and layout.

What goes in Main?

- The main element contains the primary content of the document, such as:

  • Articles
  • Blog posts
  • Product descriptions
  • Formatted text

What goes in the Footer?

- The footer element typically contains secondary or supplementary content, such as:

  • Copyright information
  • Contact information
  • Social media links
  • Site maps
  • Disclaimer or terms of use

HTML Validation

- HTML validation is the process of checking an HTML document for errors and ensuring it conforms to the HTML specification. This can be done using tools like the W3C Markup Validation Service.

Validating HTML helps:

  • Ensure accessibility
  • Improve search engine optimization (SEO)
  • Prevent errors and inconsistencies
  • Ensure compatibility across different browsers and devices
