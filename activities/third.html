Internet:

- It is a worldwide collection of interconnected network.

Web:

- It is an online systems for pages and resources that you can access through internet.

Different between Internet and Web:

- Internet is the network of a computer which is connected and Web is the the online pages you visit using that network.

ASCII and Unicode:

- ASCII (American Standard Code for Information Interchange) is a code that represents the characters like letters and numbers

- Unicode is more adcanced code that represents characters from many different languages and symbols.

UTF-8:

- UTF-8 (Unicode Transformation Format 8-bit) is the way to encoding characters in the Unicode system by using 1 to 4 bytes.

URL:

- URL ( Uniform Resources Locatore) is a web address to access a specific resources or pages on the Internet.

Domain Name:

- It is a readable part of an URL that helps us identify a website.

HTTP:

- HTTP (Hypertext Transfer Protocol) is a protocol that shows how clients and web server communicating by used for transmitting data and web pages.

HTTPS:

- HTTPS (Hypertext Transfer Protocol Secure) is a protocol or another extension of HTTP that it adds security and encryption measures to protect the data that trasmitted over the web.

IP Address:

- IP Address (Internet Protocol Address) is label through numerical that assigned to all devices connected to a network.

DNS:

- DNS (Domain Name System) is a system that translate domain name into an IP address which allowing users to access.

W3C Recommendations:

W3C (World Wide Web Consortium) is a recommendations or quidelines to ensure the accessbility of web development.

XML, XHTML, HTML, HTML5:

- XML (Extensible Markup Language) is a markup language that allowing developers to create custom tags and structuters for storage and data exchange.

- XHTML (Extensible Hypertext Markup Language) is a markup language that combines HTML and XML features.

- HTML (Hypertext Markup Language) is a markup language that usied for creating web pages.

- HTML5 (Hypertext Markup Language 5 Version) is a 5 major version markup language of HTML that introducing new features, elements and APIs.
