CSS Grid

What should you focus on during class and after class?

During class, focus on understanding how to create rows and columns using CSS Grid and how to place items within those rows and columns. After class, practice by making your own grid layouts and trying different options to see how they work.

What is the main difference between CSS Grid and CSS Flex?

Main difference between CSS Grid and CSS Flex is that CSS Grid is best for making layouts with both rows and columns. CSS Flex is best for arranging items in a single row or a single column.
