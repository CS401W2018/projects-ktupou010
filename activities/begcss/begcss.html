What is CSS?

Cascading Style Sheets (CSS) is used to format the layout of a webpage. With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

Explain the difference between inline, embedded, and external CSS.

Inline CSS refers to CSS styles written directly in the HTML element using the style attribute. And Embedded CSS refers to CSS styles written within the HTML document using the style tag, usually in the head section. Also External CSS refers to CSS styles written in a separate file with a `.css` extension, which is then linked to the HTML document using the `link` tag.

Explain why you should only use external CSS.

Using only external CSS is a best practice for several reasons:

  • Separation of Concerns: External CSS keeps presentation styles separate from HTML structure and content, making it easier to maintain and update websites.
  • Reusability: External CSS files can be linked to multiple HTML documents, allowing for consistent styling across entire websites.
  • Flexibility: External CSS makes it easy to switch between different styles or themes by simply swapping the CSS file.
  • Scalability: As websites grow, external CSS helps keep styles organized and manageable.
  • Cacheability: External CSS files can be cached by browsers, reducing page load times.

What is a selector used for?

In CSS, a selector is used to target specific HTML elements or groups of elements to apply styles. Selectors identify the elements to which the styles will be applied.

Types of Selectors:

  • Element Selectors: Target elements by their HTML tag name (e.g., p, h1, img).
  • Class Selectors: Target elements with a specific class attribute (e.g., .header, .footer).
  • ID Selectors: Target elements with a specific ID attribute (e.g., #header, #footer).
  • Attribute Selectors: Target elements with specific attributes or attribute values.
  • Pseudo-Class Selectors: Target elements based on their state or position (e.g., :hover, :active, :first-child).
